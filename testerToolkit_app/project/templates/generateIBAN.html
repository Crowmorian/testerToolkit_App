IBAN is a number used in transactions to foreign countries or banks. Here you can generate a new IBAN number from some given parameters. The Country field represents the country where the bank for the account is based. Bank code is usually represented by a four digit code (ex. 0300). The Account area code is the part of the account number itself, that comes before the dash (19-123). Not all acounts have to contain an area code, so this field is optional. Account number is the actual number that comes after the dash (19-123) The generated number is than displayed at the bottom of the screen.
